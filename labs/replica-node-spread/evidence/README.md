# Evidence directory

Raw experimental output from `trigger.sh`, `h3-falsification.sh`, and `analyze.py`.

This directory is **permanent and committed to the repository** per the user mandate that raw data must be preserved to avoid re-running the experiment.

## File naming

- `consumption-<timestamp>.jsonl` — scale-ladder samples for Consumption profile
- `dedicated-<timestamp>.jsonl` — scale-ladder samples for Dedicated D8 profile
- `h3-same-replica-<timestamp>.jsonl` — Part A of proxy validation (idempotence)
- `h3-restart-<timestamp>.jsonl` — Part B of proxy validation (new-replica detection)
- `trigger-<timestamp>.log` — orchestration log for one full run
- `<name>-summary.md` — markdown generated by `analyze.py` from the corresponding JSONL

## Editing policy

- Raw `*.jsonl` and `*.log` files are **append-only**. Do not edit after capture.
- Generated `*-summary.md` files MAY be regenerated by re-running `analyze.py` against the same JSONL.
- Adding new runs is allowed; deleting old runs is **not** allowed without an issue documenting why.
